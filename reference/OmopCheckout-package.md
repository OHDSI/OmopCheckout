# OmopCheckout: Check and Explore Summarised Results from the OMOP Common Data Model

Provides tools to validate and explore 'summarised_result' objects
produced when working with the Observational Medical Outcomes
Partnership (OMOP) Common Data Model. Checks that results are correctly
formatted and adequately suppressed, and provides utilities to explore
and understand their content.
