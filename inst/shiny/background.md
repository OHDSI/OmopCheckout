# OmopCheckout

This app allows you to review and validate [**summarised_result objects**](https://darwin-eu.github.io/omopgenerics/articles/summarised_result.html) before sharing or further analysis.

It provides:

- 🔍 **Explore Results** — interactive table view of uploaded results  
- 🧾 **Checkout Summary** — structured overview including:
  - Total rows and result IDs  
  - Suppression status  
  - Package names and versions  
  - Estimate and strata coverage  

The aim is to support transparent, reproducible, and governance-aligned result sharing.

Upload a compatible `summarised_result` `.csv` file to begin.
